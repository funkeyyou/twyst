# Twýst - 智能時尚購物平台

**Twýst** 是一個結合了 AI 生成技術的現代化時尚電子商務平台演示。本專案展示了如何利用 React 19、Tailwind CSS 與 Google Gemini API (2.5 Flash) 打造流暢、具備個人化體驗且視覺效果極佳的購物網站。

## ✨ 核心功能亮點

### 1. 🤖 AI 智能應用
*   **Lumi 個人造型師**：
    *   整合 **Gemini 2.5 Flash** 模型。
    *   提供即時的時尚穿搭建議、風格分析與禮物推薦。
    *   角色設定為「優雅、親切的高級造型顧問」，支援繁體中文對話。
*   **AI 虛擬試衣間**：
    *   利用 Gemini 的多模態影像編輯能力。
    *   使用者可上傳個人全身照（儲存於瀏覽器記憶體中）。
    *   系統將選定的商品「合成」至使用者照片上，模擬穿著效果。

### 2. 🛍️ 沉浸式購物體驗
*   **現代化首頁設計**：包含 Hero 大圖、分類導航、橫向捲動的新品推薦與熱銷排行。
*   **快速瀏覽彈窗 (Quick View)**：點擊商品不跳轉頁面，直接彈出詳細資訊視窗，保持瀏覽節奏。
*   **智慧購物袋**：
    *   **物流選擇**：支援模擬 7-11 超商取貨、宅配到府、蝦皮店到店（含運費計算與免運邏輯）。
    *   **優惠券系統**：一鍵套用可用優惠券，支援折扣碼輸入與最低消費門檻檢查。
    *   **收藏功能**：可將購物車商品「移至收藏」。

### 3. 👑 深度會員與訂單系統
*   **會員等級制度**：
    *   四階等級（一般、銀卡、金卡、鑽石）。
    *   視覺化積分進度條與專屬權益彈窗。
    *   支援第三方帳號（Google）模擬綁定。
*   **全功能訂單管理**：
    *   **訂單詳情**：視覺化物流進度條（成立 -> 出貨 -> 送達）、費用明細。
    *   **售後服務**：模擬退款申請流程（支援原因選擇與照片上傳）。
    *   **再次購買**：一鍵將歷史訂單商品重新加入購物車。

## 🛠️ 技術棧

*   **前端框架**: React 19
*   **語言**: TypeScript
*   **樣式庫**: Tailwind CSS (透過 CDN 載入，支援自訂 Config)
*   **圖標庫**: Lucide React
*   **AI 整合**: Google GenAI SDK (`@google/genai`)
*   **建置環境**: 無需 Bundler，直接使用 ES Modules 於瀏覽器運行。

## 🚀 如何體驗

1.  **瀏覽與選購**：在首頁查看新品，點擊商品卡片開啟快速預覽，將喜歡的商品加入購物袋。
2.  **AI 互動**：
    *   點擊右下角「✨」圖示諮詢穿搭建議。
    *   在商品詳情點擊「AI 虛擬試衣」，上傳照片體驗合成效果。
3.  **結帳流程**：
    *   打開購物袋，選擇「7-11 超商取貨」。
    *   選擇一張優惠券（如 `WELCOME10`）。
    *   點擊結帳（需先登入，系統會自動模擬金卡會員登入）。
4.  **會員管理**：
    *   結帳後進入「會員中心」。
    *   查看剛建立的訂單，點擊「查看詳情」追蹤物流。
    *   試著對已完成的訂單申請退款。
